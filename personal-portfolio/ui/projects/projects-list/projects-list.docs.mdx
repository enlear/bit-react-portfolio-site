A component that displays a list of projects.
Use with `ThemeProvider` to enforce thememing.

### Component usage

```js
<ProjectsList max={10} />
```

### Render project list with no maximum number

```js live
<MemoryRouter>
  <ProjectsList />
</MemoryRouter>
```
